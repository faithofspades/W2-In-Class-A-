Data Types

Data types in programming define the type of value a variable can hold, determining how the computer interprets and processes that value.


Go to the JS Console and type

typeof 42


typeof "hello"


typeof false
  • Numbers: Represent both integers and floating-point values. Example: 42, 3.14. JavaScript uses 64-bit floating-point format for all numbers. 
    const age = 25;
const pi = 3.14;

  • Strings: Used for text, enclosed in single quotes ('), double quotes ("), or backticks (`). Example: "Hello", 'World'. Strings are immutable, meaning their content cannot be changed. 
    const greeting = "Hello, World!";

  • Booleans: Represent true/false values, commonly used for conditions and logical operations. Example: true, false. 
    const isEven = (5 % 2 === 0); // false


Strings and Sting Literals


  • Basic Syntax: Use backticks (`) to create string literals, which allow for embedding expressions and multiline strings. 
    const greeting = `Hello, World!`;

  • Embed Expressions with ${}: Use the ${} syntax to interpolate variables or expressions directly into a string. 
    const name = "Hooter T. Owl";
const message = `Welcome, ${name}!`;

  • Multiline Strings: String literals can span multiple lines without needing special characters like \n. 
    const multiline = `This is line one.
This is line two.`;

  • Supports Nested Expressions: Embed any JavaScript expression inside ${} for dynamic content. 
    const a = 5, b = 3;
const result = `The sum of ${a} and ${b} is ${a + b}.`;
